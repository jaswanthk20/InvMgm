Features

Everything a makerspace needs

Built lean with no over-engineering. Every feature solves a real operational problem.

🏠

Real-Time Dashboard KPIs

Out-of-stock counts, low-stock alerts, open purchase orders, and weekly stock movements — all visible on first load with no page reloads.

⚡

Quick Transactions (≤ 2 Clicks)

Submit IN or OUT stock adjustments from the main dashboard without navigating away. A modal captures item, location, quantity, and movement type.

📦

Item Master List

Add, edit, or deactivate items. Dual-layer filtering: free-text search combined with a live-generated category dropdown derived from actual data — no hardcoded lists.

📥

Bulk CSV / Excel Import

Drag-and-drop upload on the Items page. Automatically detects and skips duplicate SKUs, mapping spreadsheet columns to the item master schema.

🛒

Purchase Orders & Receiving

Create order requests per item. Items enter PENDING state and surface on the dashboard. Mark as Received to close the order and auto-trigger a stock IN movement.

📋

Append-Only Audit Ledger

Every stock movement is written to the stock_movements table. Records are never deleted or updated — providing a full, tamper-proof paper trail.

Getting Started

Up and running in minutes

No Docker, no cloud accounts, no build step. Just Python and pip.

1

Clone the repository

Get the source code onto your machine.

$ git clone https://github.com/jaswanthk20/InvMgm.git
$ cd InvMgm
2

Create a virtual environment

Isolate dependencies from your system Python.

$ python -m venv venv
$ venv\Scripts\activate
3

Install dependencies

All packages listed in requirements.txt.

$ pip install -r requirements.txt
4

Initialize the database

Creates database.db by executing schema.sql.

$ flask --app app init-db
5

(Optional) Seed sample data

Populate with realistic locations, items, and metric logs.

$ python seed_data.py
6

Start the development server

Open http://localhost:5000 in your browser.

$ flask --app app run --port 5000
API Reference

Clean REST endpoints

All endpoints return JSON. Base URL: http://localhost:5000

Method Endpoint Description
GET /api/items List all items
POST /api/items Create a new item
PUT /api/items/<id> Update an existing item
DELETE /api/items/<id> Remove an item
GET /api/locations List all locations
POST /api/locations Create a new location
POST /api/transactions Log a stock IN / OUT / MOVE
GET /api/dashboard/metrics Returns KPI counts (low stock, open POs, weekly movements)
GET /api/dashboard/low-stock Returns items below their min_stock threshold
GET /api/orders List all purchase orders
POST /api/orders Create a new purchase order
PUT /api/orders/<id>/receive Mark as received — triggers stock IN automatically
POST /api/import Upload CSV or Excel file for bulk item registration
